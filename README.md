# 📈 Stock Pulse – Real-Time Stock Market Analyzer

## Overview

Stock Pulse is a real-time stock market analysis application built using Python and Streamlit. It enables users to monitor live and historical stock data, analyze market trends using technical indicators, and visualize insights through interactive charts. The application is designed to help investors and learners make data-driven decisions with an intuitive and user-friendly interface.

## Features

* Real-time stock price tracking
* Historical stock data analysis
* Technical indicators such as Moving Averages and RSI
* Interactive charts using Plotly
* Secure data storage with SQLite
* Easy-to-use Streamlit dashboard
* Fast and responsive user interface

## Tech Stack

* **Programming Language:** Python
* **Framework:** Streamlit
* **Libraries:** Pandas, NumPy, Plotly, yfinance
* **Database:** SQLite
* **Version Control:** Git & GitHub

## Project Workflow

1. Fetch real-time and historical stock market data.
2. Store stock information in a SQLite database.
3. Calculate technical indicators for trend analysis.
4. Generate interactive visualizations using Plotly.
5. Display insights through the Streamlit dashboard.

## Installation

1. Clone the repository.
2. Install the required dependencies:

   ```bash
   pip install -r requirements.txt
   ```
3. Run the application:

   ```bash
   streamlit run app.py
   ```
